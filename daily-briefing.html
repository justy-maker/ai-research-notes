馬斯克 xAI 宣布重組並揭露月球計畫

OpenAI 推出 GPT-5.3-Codex 和企業級代理平臺 Frontier

OpenAI 發布了 GPT-5.3-Codex，這是一個專門用於程式碼生成的模型，代表著人工智慧開始協助打造人工智慧的新時代。同時，OpenAI 也推出了名為 Frontier 的代理管理平臺，專門針對企業級的人工智慧協作需求。此外，OpenAI 也擴充了 Responses API 的功能，新增了伺服端壓縮整理與 Shell 容器，支援長時間運行的任務。

Anthropic 與 OpenAI 在政治捐贈上的不同立場

Google 揭露國家級駭客大量採用人工智慧

Google 的資安報告揭露，北韓、中國和伊朗的駭客組織已經全面導入人工智慧技術進行攻擊。報告指出，模型竊取攻擊正在激增，人工智慧已經成為駭客的新武器。

Google AI 健康資訊準確性遭質疑

外媒報導指出，Google 的人工智慧摘要功能在健康資訊方面存在錯誤，嚴重情況下可能危害使用者的生命安全。專家提醒民眾在使用人工智慧獲取健康資訊時要格外謹慎，不要完全依賴。

Chrome 開始支援 WebMCP 協議

Google Chrome 瀏覽器開始支援 WebMCP 早期預覽版，這讓網站可以將功能公開成人工智慧代理工具。未來人工智慧代理將可以直接與網頁元件互動，開啟了代理式網頁應用的新可能。

Cloudflare 推出 AI 代理友善的網頁轉換服務

Cloudflare 推出 Markdown for Agents 服務，人工智慧代理只需要在請求標頭中要求 Markdown 格式，就能取得即時轉換的版本，可以減少百分之八十的詞元用量，大幅降低人工智慧的運算成本。
