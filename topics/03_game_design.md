# 遊戲設計研究

> 為兒子的遊戲開發夢想準備 🎮

## 📋 研究目標

探索遊戲設計與開發的學習路徑，整理適合入門的資源與工具。

## 🔍 研究方向

1. **遊戲引擎**
   - Unity（C#）
   - Unreal Engine（C++/Blueprint）
   - Godot（GDScript/C#）
   - GameMaker

2. **遊戲設計基礎**
   - 遊戲機制設計
   - 關卡設計
   - 敘事設計
   - 玩家心理學

3. **美術與音效**
   - 像素藝術
   - 3D 建模
   - 音效設計
   - AI 輔助創作

4. **AI 在遊戲開發的應用**
   - AI 生成遊戲素材
   - AI NPC 行為
   - 程式碼輔助

## 🎯 學習路徑建議

### 初學者（1-3 個月）
1. 選擇一個引擎（推薦 Unity 或 Godot）
2. 完成官方教學
3. 做一個簡單的小遊戲

### 進階（3-6 個月）
1. 學習遊戲設計理論
2. 參加 Game Jam
3. 做一個完整的小作品
