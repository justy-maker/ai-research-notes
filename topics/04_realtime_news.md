# 📰 如何蒐集即時的財經與科技快訊

> ✅ **已解決** - 建立了自動化新聞監測系統

---

## 🎯 解決方案

### 系統架構

```
┌─────────────────────────────────────────┐
│           每小時自動執行                  │
│         (08:00-22:00 GMT+8)              │
└─────────────────┬───────────────────────┘
                  ▼
┌─────────────────────────────────────────┐
│  Gemini Deep Search + web_fetch         │
│  ├── TechCrunch AI                      │
│  ├── The Verge AI                       │
│  ├── CNBC Technology                    │
│  ├── xcancel.com (Twitter 替代)         │
│  └── Reuters Technology                 │
└─────────────────┬───────────────────────┘
                  ▼
┌─────────────────────────────────────────┐
│  AI 篩選重要新聞（1-5 則）               │
│  根據追蹤目標過濾                        │
└─────────────────┬───────────────────────┘
                  ▼
┌─────────────────────────────────────────┐
│  發送到 Discord #重要新聞               │
└─────────────────────────────────────────┘
```

### 追蹤目標

| 類別 | 追蹤對象 |
|------|----------|
| **人物** | 川普、馬斯克、黃仁勳、奧特曼 (Sam Altman) |
| **公司** | Google、OpenAI、Anthropic、TSMC、Tesla、Nvidia |

### 篩選標準

- 🔴 股市大波動（>2%）
- 🏦 央行政策變化
- 🚀 大廠重大發布
- 🤖 AI 重大突破
- 🎤 重要人物聲明

---

## ⚙️ 技術配置

| 項目 | 設定 |
|------|------|
| Cron Job | `hourly-news` |
| 執行時間 | 每小時 08:00-22:00 (GMT+8) |
| 搜索引擎 | Gemini CLI (Google AI) |
| 輸出頻道 | Discord #重要新聞 |
| wakeMode | `now` |

### 新聞來源

| 來源 | 類型 | 狀態 |
|------|------|------|
| TechCrunch | AI/科技 | ✅ 穩定 |
| The Verge | AI/科技 | ✅ 穩定 |
| CNBC | 財經 | ✅ 穩定 |
| Reuters | 綜合 | 🟡 偶爾被擋 |
| xcancel.com | Twitter | ✅ 部分帳號可用 |
