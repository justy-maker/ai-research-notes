## 實際應用示例

### 1. CRM 表單自動填寫
```
導航到表單 → 視覺理解頁面布局 → 讀取表單欄位
→ 提取寵物和主人詳情 → 篩選加州居民
→ 登入系統 → 映射到正確 CRM 欄位
→ 創建客戶檔案 → 驗證添加成功
→ 預約時段 → 安排後續會議
```
**全程通過 UI 操作，無需 API 或自定義整合**

### 2. 便利貼自動分類
- 打開數位白板
- 視覺掃描所有便利貼
- 理解文字內容
- 自動分類（推廣、設置、志願者）
- **拖放**到正確位置

### 3. GitHub PR 審查
- 導航到 Browserbase Stagehand 項目
- 找到最新的非草稿 PR
- 驗證 PR 是否通過
- **速度極快**（約 10 秒完成導航）

### 4. YouTube 頻道搜索
- 導航到指定頻道
- 點擊「Popular」標籤
- 找到最受歡迎的影片
- 其他模型無法正確點擊的操作，Gemini 3.0 可以

### 5. 大學 AI 活動抓取
```
任務：找到未來 60 天的 AI 相關活動
→ 多頁面導航
→ 語義推理判斷「AI 相關」
→ 處理 PDF、活動頁面、日曆
→ 提取標題、日期、時間、地點、虛擬連結
→ 輸出 JSON + HTML 表格
→ 按日期排序
```

## 使用方式（全部免費！）

### 1. Browserbase
- AI 瀏覽器自動化框架
- Google 合作夥伴
- 可本地部署

### 2. Google AI Studio
- Build Mode 或 Studio 直接使用
- 適合需要 Computer Use 的任務

### 3. Antigravity（推薦）
- Google 的免費 IDE
- **Agent Manager** 介面
- 即時預覽 Agent 操作
- 可即時溝通調整

### 4. Stagehand
- 開源工具
- 使用 Gemini Computer Use
