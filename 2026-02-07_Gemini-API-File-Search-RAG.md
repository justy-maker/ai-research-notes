## 核心論點
Google DeepMind 推出 Gemini API 的「File Search Tool」，是一個全託管 RAG 系統，徹底簡化 AI 模型與自有資料整合的複雜流程，且計費模式極具成本優勢。

## 重點整理

### 功能特色
1. **全託管 RAG 系統** - 直接整合進 Gemini API
2. **自動化處理** - 檔案儲存、最佳分塊策略、嵌入生成全自動
3. **動態上下文注入** - 檢索到的資訊自動注入回應
4. **引文支援** - 回應自動附上資訊來源，確保可驗證性

### 超低成本計費
- 🆓 **儲存免費** - 檔案儲存不收費
- 🆓 **查詢嵌入免費** - 查詢時的嵌入生成免費
- 💰 **首次索引** - 每 100 萬 tokens 僅 $0.15

### 支援格式
- PDF、DOCX、TXT
- 各種程式語言檔案
- 適合建立全面知識庫

### 應用場景
- 智慧客服機器人
- 內部知識助理
- 內容發現平台
