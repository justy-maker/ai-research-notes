## MCP 指令操作

| 指令 | 功能 |
|------|------|
| `List Projects` | 拉取帳號下所有項目 |
| `List Screens` | 列出項目內所有頁面 |
| `Get Project` | 獲取項目資訊 |
| `Create Project` | 從零創建新項目 |

### 使用範例
```
「Stage 列出項目」→ 觸發 MCP 工具 → 返回所有項目
「列出該 ID 對應的所有頁面」→ 拉取項目內頁面
```

技能是一種**技差集用**的能力，賦予 Agent 後可實現設計轉代碼。

| 技能 | 功能 |
|------|------|
| **Design AMD** | 拆解設計系統 |
| **React Components** | 設計稿 → 生產級 React 組件 |
| **Enhance Prompt** | 粗略想法 → 完善 UI 提示詞 |
| **Stitch Loop** ⭐ | 一個提示詞 → 完整多頁面網站 |

## 實戰流程

### 流程 1：現有設計 → React 項目

```
1. List Projects → 選擇項目
2. List Screens → 獲取頁面（如：商品列表、購物車、結算）
3. 輸入：「使用所有頁面設計創建 React 應用」
4. 自動生成：
   - 完整項目結構
   - 規範的組件和 Hooks
   - 直接轉換設計稿 HTML
   - 精準複用原始設計資源
```

### 流程 2：從零創建（Stitch Loop）

```
1. 用 Claude 生成計劃（如：現代健身網站方案）
2. 詳細拆解主題風格和頁面布局
3. 輸入：「利用 Stitch Loop 技能創建 Stitch 項目」
4. 自動逐一生成所有頁面（本例：6 個頁面）
5. 自動創建 Design.md 和 Site.md 文件
```
