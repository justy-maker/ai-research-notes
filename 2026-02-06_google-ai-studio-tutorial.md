## 📝 一句話摘要

Google AI Studio 是一個免費平台，只需 Google 帳號就能使用 Gemini 3.0 Pro、圖片生成、影片生成、TTS 等功能，還能用自然語言開發 APP。

## 🎯 核心內容

### 平台介紹

- **免費功能**：對話、基本圖片生成、文字轉語音（TTS）
- **付費功能**：Nano Banana Pro、Veo 3.1
- **優勢**：Gemini 3 Pro 在官方每天只能用 3 次，AI Studio 可用 50-225 次

### 重要參數設定

| 參數 | 說明 |
|------|------|
| 系統指令 | 設定 AI 角色、風格、回覆格式 |
| 溫度 (0-2) | 0=保守穩定，2=創意隨機，日常 0.7-1 |
| 媒體分辨率 | 高=細節多但慢，低=快但粗略 |
| 思考水平 | 控制 AI 推理深度 |

### 文件上傳限制

- 只支援 PDF 格式
- Word/Excel 需先轉 PDF，或從 Google Drive 導入

## 🛠️ 五大實戰案例

### 案例 1：爆款影片分析

**用途**：分析 YouTube 爆款影片的共同規律

**技巧**：
- 上傳影片連結讓 Gemini 3 Pro 分析
- 指定時間範圍（如前 2 分鐘）節省 Token
- 輸出：標題公式、開頭鈎子、文章結構、金句模板

**發現的爆款公式**：
- 權威背書 + 名人效應（巴菲特、川普）
- 極致數字反差（暴跌 10%、3300 億）
- 製造焦慮痛點（階級固化、財富洗牌）

### 案例 2：復刻爆款短影片

**流程**：
1. 下載原影片，上傳給 Gemini 3 Pro 分析
2. 生成圖片提示詞（含人物、服裝、背景、光線、構圖）
3. 用 Imagen 4 生成圖片（9:16 尺寸）
4. 用 Veo 2 轉成影片（8 秒）

### 案例 3：Build 功能開發 APP

**範例**：YouTube 縮圖全平台適配工具

**步驟**：
1. 用 Gemini 將需求翻譯成專業提示詞
2. 進入 Build 界面，選擇 Nano Banana 模型
3. 貼上提示詞，選 Gemini 3 Pro，點擊生成
4. 測試、調整、分享連結

**特點**：
- 不只裁剪，會智能重新排版
- 文字和元素自動適配新尺寸
- 可部署到 Google Cloud 商業化

### 案例 4：Gemini Live 即時協作

**用途**：設計或剪輯時即時獲得 AI 建議

**操作**：
- 分享螢幕，AI 即時分析畫面
- 可中斷、追問、實時修改
- 適合：影片剪輯、圖片設計、網頁開發

**注意**：Token 消耗較高，可能較快達到上限

### 案例 5：TTS 文字轉語音

**功能**：
- 30+ 種聲音選擇
- 可設定語氣（專業、溫和等）
- 無限次數生成

**操作**：
1. 進入 Playground → Audio
2. 選擇 2.5 Pro 模型
3. 選擇聲音、設定語氣
4. 貼上腳本，生成並下載
