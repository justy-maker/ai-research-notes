## 核心觀點

介紹 Figma 新推出的 Site 功能，可以直接在 Figma 中製作並發佈網站。注意：Figma Site 需要付費訂閱才能使用。

### 快速預覽
- 總流程概覽
- 開新檔案與 Breakpoints 設定
- Responsive 設計大方向概念

### Responsive 設計
- 製作 Responsive 設計物件
- Site 自動調整 Responsive 物件
- Primary 板塊順序層級
- Design 到 Site 前端語法差異

### 設計添加
- 內建範例與自己的 Library
- 文字斷點設定（Text Style Breakpoints）
- 設計優化技巧
- Canvas Stacking 與響應設計調整

### 互動與發佈
- Interaction 互動設計
- 新增多個網頁分頁（Webpage）
- 網站設定（Title、Favicon 等）
- Accessibility 無障礙功能

### 額外技巧
- 縮短 URL（tinyURL）
