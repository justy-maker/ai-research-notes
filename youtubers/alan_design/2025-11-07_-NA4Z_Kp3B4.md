## 主題
使用 Figma 製作麵包屑導覽 (Breadcrumb Navigation)

## 關鍵步驟

* **建立基礎按鈕元件 (Base Tab)：** 使用 Text 工具創建文字連結，添加 Auto Layout 調整間距和底色，並建立 Component，再加入 Variant (Hover State)。
* **設定 Hover 效果：** 為 Hover State 設定顏色，使其在滑鼠懸停時有視覺反饋。
* **建立文字屬性 (Text Property)：** 使用 Multi-edit 功能，為按鈕文字建立 Property，方便後續快速修改文字內容。
* **建立麵包屑元件 (Breadcrumb)：** 複製基礎按鈕元件，添加分隔符號 (例如斜線)，並使用 Auto Layout 調整間距，建立 Component。
* **建立不同層級的麵包屑變體 (Variant)：** 複製麵包屑元件，建立不同層級的 Variant，顯示不同數量的連結，並根據需求調整顯示方式（例如隱藏過多層級）。

## 實用技巧

* 將常用顏色 (例如灰、白、黑) 添加到顏色樣式庫 (Color Style Library)，方便統一管理。
* 使用 Instance 複製元件，保持設計一致性。
* 考量手機螢幕的顯示，避免麵包屑導覽過於複雜，導致斷行或重疊。
