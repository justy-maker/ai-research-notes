# Don't Beat Them. Learn From Them.

- **頻道**: Simon Sinek
- **影片 ID**: W36s9C1kTHw
- **連結**: https://www.youtube.com/watch?v=W36s9C1kTHw
- **時長**: 約 3 分鐘

## 摘要

Simon Sinek 用 Microsoft 與 Apple 的對比，說明無限賽局中不需要「打敗對手」，而是向「值得尊敬的對手」學習。

### 核心觀點

#### 值得尊敬的對手（Worthy Rivals）
- 在無限賽局中沒有贏家或輸家
- 比你強的公司是「禮物」——他們的優點揭示你的弱點
- 當你一路領先時，容易對自己的缺點視而不見

#### Microsoft vs Apple 的故事
**Microsoft 教育峰會**：
- 80% 的高層簡報都在講「如何打敗 Apple」

**Apple 教育峰會**：
- 100% 的高層簡報都在講「如何幫助老師教學、幫助學生學習」

結果：哪一家有士氣、創新、合作的問題？

#### Zune vs iPod 的故事
- Microsoft 送了 Simon 一台 Zune（對標 iPod 的產品）
- Simon 在計程車上跟 Apple 高層說：「Zune 比我的 iPod 好太多了」
- Apple 高層回應：「我完全不懷疑」——然後話題就結束了
- 因為具有無限思維的人知道：有時領先、有時落後，這不叫贏或輸
- 兩年後，Apple 推出 iPhone，讓 Zune 和 iPod 都變成過時產品

### 實際應用

- 專注在自己身上，專注在如何成為更好版本的自己
- 向比你強的對手學習——他們是值得尊敬的對手
- 跑步時有配速員會跑更快，但你不是要「打敗」配速員

## 金句

> 「目標不是打敗你的競爭對手。目標是超越自己。」
