# True Kindness Is Uncomfortable

- **頻道**: Simon Sinek
- **影片 ID**: a-Qhz-X8Ws0
- **連結**: https://www.youtube.com/watch?v=a-Qhz-X8Ws0
- **時長**: 約 3 分鐘

## 摘要

Simon Sinek 區分「親切」(nice) 與「善意」(kind) 的差異，並指出真正的善意必然帶來不適感。

### 核心觀點

#### 親切 vs 善意
- **親切**：善意的「表演」，但不一定有行動
- **善意**：真正採取行動

**例子**：
- 給人直接的回饋 → 善意但不親切
- 不給人回饋（怕對方不高興）→ 親切但不善意
- 朋友臉上有東西卻不告訴他 → 親切但不善意

#### 北歐人的例子
- 瑞典人通常「不親切但很善意」
- 這是個複雜的概念

#### 真正的善意是不舒服的
- **親切**：讓大家感覺一切順利，沒有不適
- **善意**：需要擁抱不適、面對困難

對自己善意也是不舒服的：
- 必須把自己放在第一位
- 必須給自己寬容
- 有時打擊自己比較容易

#### 善意 vs 慷慨
- 慷慨：捐錢給慈善機構
- 善意：付出「不可贖回的資源」——時間與精力

**搬家的例子**：
- 慷慨：「我幫你付搬家車的錢」
- 善意：「我來你家幫你打包、搬運、拆箱」

#### 善意需要勇氣與犧牲
- 善意帶有某種程度的「犧牲」
- 付出的是無法拿回來的東西

## 金句

> 「真正的善意是不舒服的。親切則不會讓人不舒服。這就是差別。」
