## 重點整理

1. **自動化 PDF 資料提取**：影片核心在於展示如何使用 n8n 整合 Unstract 工具，將 PDF 文件（如發票、收據）中的非結構化資料，自動轉換為結構化的 JSON 格式，無須手動複製貼上。

2. **整合多元應用**：透過 n8n 的工作流程，可以輕鬆串接 Gmail、Google Sheets、Discord 等超過 350 種應用程式，實現從接收郵件、提取附件、處理資料到存入表格的完整自動化流程。

3. **開源且免費自架**：n8n 和 Unstract 都是開源工具，可以免費在本地端或自己的伺服器上運行，確保了資料的隱私與安全，同時降低了使用成本。

4. **LLM 技術驅動**：此解決方案利用大型語言模型（LLM）來理解與解析文件內容，能精準辨識欄位（如發票號碼、總金額、日期），大幅提升資料提取的準確性與彈性。

5. **應用場景廣泛**：除了處理發票，這套方法也適用於自動化數據輸入、處理表單提交、建立聊天機器人等，能有效提升各種重複性文件處理工作的效率。

## 內容摘要

這部影片詳細介紹如何結合 n8n 與 Unstract 這兩款強大的開源工具，輕鬆建立自動化的 PDF 資料提取工作流程。傳統上，從 PDF 文件（例如供應商發票）中手動複製資訊到電子表格，不僅耗時且容易出錯。影片所提出的解決方案，能將整個過程完全自動化：從指定的來源（如 Gmail 附件）自動抓取 PDF 文件，接著利用 Unstract 的大型語言模型（LLM）技術，精準地讀取並解析文件內的關鍵資訊，例如公司名稱、發票總額、日期等，並將這些非結構化資料轉換成標準的 JSON 格式。

n8n 作為一個視覺化的工作流程編輯器，讓使用者能輕易地將 Unstract 與其他應用程式（如 Google Sheets、Airtable 或資料庫）串連起來。這意味著提取出來的結構化資料，可以無縫地自動寫入指定的表格或系統中，完成端到端的自動化。此解決方案完全免費，並且可以在本地環境自行架設，確保了資料的隱私與安全。對於需要處理大量發票、收據、合約等文件的企業或個人來說，這套方法能顯著節省時間、降低人力成本，並大幅提升數據處理的準確性與效率。
