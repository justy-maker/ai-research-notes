# OpenClaw + OpenCode Turns Your AI Into a Full Autonomous Engineering Team!

- **頻道**: In The World of AI
- **日期**: 2026-02-14
- **連結**: https://www.youtube.com/watch?v=kIWMLL0S8X8
- **語言**: EN

## 重點摘要

影片介紹如何結合 OpenClaw（自主AI代理）和 Open Code（開源AI編碼代理）打造更強大的 AI 工作流程。OpenClaw 負責任務規劃和分配，Open Code 則執行任務，能讀取、編輯和理解程式碼，並使用多個子代理。影片也提及 OpenClaw 的安全性問題，並介紹了 Gen Digital 的 Agent Trust Hub，可掃描並阻擋惡意技能。此外，說明了使用 Google 驗證在 OpenClaw 中免費使用最先進模型的方法。

## 主要內容

- **OpenClaw 介紹：** 一個在電腦上運行的自主 AI 代理，可作為個人AI助手
- **Open Code 介紹：** 一個開源 AI 編碼代理，可以直接讀取、編輯和推理程式碼，並能使用各種模型
- **結合 OpenClaw 和 Open Code：** OpenClaw 負責任務規劃和分配，Open Code 則負責執行，OpenClaw 變成指揮中心
- **OpenClaw 安全性問題：** 指出社群提供的 OpenClaw 技能可能存在惡意程式碼，可能洩露個人資訊
- **Agent Trust Hub 介紹：** 由 Gen Digital 推出，用於掃描和阻擋 OpenClaw 技能中的惡意威脅
- **使用 Google 驗證：** 讓使用者能在 OpenClaw 中免費使用 Google 的先進模型
- **實際應用案例：** 展示了如何使用 OpenClaw 和 Open Code 創建一個功能性的 SaaS 應用程式

## 關鍵觀點

- 結合 OpenClaw 和 Open Code 可以創建更強大、更靈活的 AI 工作流程
- OpenClaw 的安全性是一個重要問題，使用安全工具進行掃描是必要的
- Agent Trust Hub 有助於使用者安全使用 OpenClaw 技能
- 利用 Google 驗證，能以更低的成本使用先進的 AI 模型
