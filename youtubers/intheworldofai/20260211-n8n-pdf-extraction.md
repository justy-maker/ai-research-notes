## 影片主題

使用 Unstruct 和 N8N 建立免費、本地的 AI 自動化工作流程，從 PDF 等文件中提取結構化資訊。

## 主要重點

- 介紹如何結合 Unstruct（結構化文件提取工具）和 N8N（AI 工作流程工具）來自動化 PDF 數據提取
- Unstruct 能夠將非結構化文檔轉化為結構化數據，而 N8N 可以建立多步驟 AI 代理自動化流程
- 展示了 Unstruct 的 Playground，可免費上傳文件並使用 AI 模型轉錄和結構化數據
- 示範了如何設置 N8N，包括使用 NPX 命令或 Docker 部署，以及如何獲取免費許可證以使用高級功能
- 介紹了如何在 N8N 中啟用 Unstruct 的自定義節點，以便訪問 Unstruct 的 API
- 描述了如何使用 N8N 編輯器建立自動化工作流程，例如從 Gmail 提取發票數據並將其輸出到 Google Sheets
- 建議建立基於表單提交觸發的聊天機器人，以便通過聊天機器人處理文件

## 技術工具提及

| 工具 | 用途 |
|------|------|
| **Unstruct** | 開源工具，將非結構化文檔轉換為結構化數據 |
| **N8N** | 開源 AI 工作流程工具，創建多步驟 AI 代理自動化 |
| **Whisper** | Unstruct 使用的 LLM 引擎 |
| **Node.js (NPX)** | 快速啟動 N8N |
| **Docker** | 容器化部署 N8N |
| **Gmail** | 觸發自動化工作流程 |
| **Google Sheets** | 輸出提取的數據 |
