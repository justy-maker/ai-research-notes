# GPT-5.3 Codex Is INSANE! OpenAI's BEST Model Might Beat Opus 4.6? (Fully Tested)

## 摘要

影片主要探討了 OpenAI 最新發布的 GPT-5.3 Codex 模型，並將其與 Anthropic 的 Opus 4.6 進行比較。影片強調 GPT-5.3 Codex 不僅僅是一個更好的編碼模型，它更快速、更聰明，且更具自主性，能夠處理長時間運行的任務，更像是一個協作夥伴而非單純的工具。影片提到 GPT-5.3 Codex 在 Swaybench Pro 和 Terminal Bench 等基準測試中表現出色，顯示其在作業系統環境、程式碼生成以及現實世界任務執行方面的強大能力。影片作者將親自測試這兩個模型，比較它們的速度、輸出結果，以判斷哪個模型才是目前最佳的編碼模型。

影片作者使用 Bolt 沙盒環境來測試 GPT-5.3 Codex，並給予它一些具有挑戰性的提示，以驗證其是否如基準測試所顯示的那樣強大。作者提到，OpenAI 允許透過 API 使用 GPT-5.3 Codex，這意味著使用者可以創建自己的編碼代理或將其與 MCP 伺服器等系統整合。此外，OpenAI 還提到了一種名為 "Tasks" 的功能，它就像 Codex 內部的自動化任務執行器。影片將著重於比較 GPT-5.3 Codex 與 Opus 4.6，以確定哪個模型在編碼方面更勝一籌。

## 重點

- OpenAI 發布了 GPT-5.3 Codex，這是他們迄今為止最強大的 AENTIC 編碼模型。
- GPT-5.3 Codex 不僅僅是編碼能力更強，它還更快、更聰明、更自主。
- GPT-5.3 Codex 的速度提升了約 25%，使其適用於涉及研究、工具使用和多步驟執行的長期複雜工作流程。
- GPT-5.3 Codex 在 Swaybench Pro 和 Terminal Bench 等基準測試中創下了新的行業標準。
- OpenAI 允許通過 API 使用 GPT-5.3 Codex，可以創建自定義編碼代理或與其他系統整合。
- OpenAI 提到了一種名為 "Tasks" 的功能，這是一個在 Codex 內部的自動化任務執行器。
- 影片將 GPT-5.3 Codex 與 Anthropic 的 Opus 4.6 進行全面比較，以確定哪個模型是目前最佳的編碼模型。
- 測試將在 Bolt 沙盒環境中進行，並給予模型具挑戰性的提示，以驗證其能力。
