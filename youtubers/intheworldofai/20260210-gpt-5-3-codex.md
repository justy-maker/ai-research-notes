## 詳細內容
OpenAI 發布了 GPT-5.3 Codex，這是一個具有強大編碼能力的 AI 模型，其速度比之前的 Codex 5.2 快 25%，且能處理長期的複雜工作流程。它在多項基準測試中表現出色，尤其是在網路開發方面，能夠從零開始構建功能完整的遊戲和應用程式，並能更好地理解使用者的意圖，產出更完整的程式碼。

GPT-5.3 Codex 除了編碼之外，還支援軟體和知識工作生命週期的各個階段，包括除錯、部署和撰寫文件。它能夠根據提示提供財務建議、產出訓練文件，並可以處理試算表和製作簡報，這表明 OpenAI 正在與 Anthropic 的 Claude 模型競爭。

目前，GPT-5.3 Codex 只能透過 Codex 應用程式、Chat GPT、Codex CLI 或 VS Code 擴充功能存取，API 尚未開放。在 Counterstrike 風格的基準測試中，GPT-5.3 Codex 的速度是 Opus 4.6 的兩倍，但 Claude 在大多數提示中做出了更好的決策和產出了更高品質的結果。
