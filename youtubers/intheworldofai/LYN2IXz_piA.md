# Claude Opus 4.6: Greatest AI Coding Model Ever! 1M Context, Agentic, & More!

## 摘要

這段影片主要介紹了 Anthropic 最新推出的 AI 模型 Claude Opus 4.6。Opus 4.6 是目前 Anthropic 最聰明的模型，相較於之前的版本，它在規劃能力、維持長期任務、處理大型程式碼庫以及自我糾錯方面都有顯著提升。最引人注目的是它擁有 100 萬 token 的上下文窗口，這使得它在處理複雜和長時間的任務時更加出色。

除了程式碼生成，Opus 4.6 也被定位為適用於日常知識工作的工具，例如財務分析、研究、建模以及處理文件、電子表格和簡報。在多項評估中，Opus 4.6 在 Agentic 編碼方面表現出色，並在 ARC AGI 2 上取得了新的最佳成績。它採用了混合推理模式，可以在深度思考、快速生成和簡單檢索之間切換，從而提高效率並降低成本。Opus 4.6 在程式碼生成方面表現出色，尤其擅長生成長時間且功能完整的程式碼，並且在長上下文處理、指令遵循和 Agentic 迴圈中也得到了改進。

## 重點

- Anthropic 發布了 Claude Opus 4.6，這是他們目前最聰明的模型。
- Opus 4.6 擁有 100 萬 token 的上下文窗口，這是一個顯著的改進。
- 除了程式碼生成外，該模型還適用於日常知識工作，例如財務分析和研究。
- Opus 4.6 在 Agentic 編碼和多學科推理方面表現出色，並在 ARC AGI 2 上取得了新的最佳成績。
- 該模型在 Terminal Bench 2.0 和 Sway Bench Pro 3 上名列前茅，並在其他推理基準測試中表現更好。
- Opus 4.6 採用混合推理模式，可以在深度思考、快速生成和簡單檢索之間切換，從而提高效率。
- 該模型擅長生成長時間且功能完整的程式碼，並且在長上下文處理、指令遵循和 Agentic 迴圈中也得到了改進。
- 雖然速度不如 Sonet 或 Haiku，但 Opus 4.6 仍然相當快速。
