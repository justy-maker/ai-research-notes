# Claude Code NEW Update IS HUGE! Claude Code Security, Claude Engineer, & MORE!

## 重點摘要

Anthropic 對 Claude Code 進行了兩項重大更新：

1. **Claude Code Desktop 更新**
   - 新增伺服器預覽功能：可在桌面介面內直接啟動開發伺服器、預覽應用程式
   - 自動讀取 console logs、捕捉錯誤並即時迭代修復
   - 支援本地程式碼審查：可在 Claude Code 內直接預覽與標註
   - PR 監控與自動修復：開啟 Pull Request 後，Claude 會在背景追蹤 CI，自動修復失敗並提交
   - 自動合併：啟用 automerge 後，CI 通過即自動合併 PR
   - Session 跨裝置移動：桌面、CLI、行動裝置間無縫切換

2. **Claude Code Security（預覽版）**
   - 自動掃描整個程式碼庫的安全漏洞
   - 像人類安全研究員一樣理解元件互動、追蹤資料流
   - 多階段驗證 + 人機協作審查
   - Dashboard 整合：可直接在 Claude Code 內檢視發現、批准修復

3. **Git Worktree 支援**
   - 多個 Agent 可平行運作，各自擁有獨立工作樹
   - 可在不同分支和任務上安全獨立工作

**效能數據**：Meta 研究估計 Claude Opus 4.6 在真實軟體任務上，50% 時間範圍內達到約 14.5 小時——人類開發者需要一整天的任務，AI 約半數時間就能完成。

## 關鍵觀點
