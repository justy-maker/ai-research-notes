## 重點摘要
- Pencil.dev 是完全免費的無限設計畫布，可作為 Figma 替代方案，與 Claude Code 深度整合
- 新功能包括：動態設計系統元件、可重用樣式、原生 AI 代理整合
- 支援多種設計系統（ShadCN、Lunarus 等），可動態減少 token 浪費並提升一致性
- 可從 Figma 直接複製設計到 Pencil，完美保留佈局、間距和樣式
- 透過 MCP 連接 Claude Code，能直接從設計畫布生成前端程式碼，實現設計到程式碼的無縫轉換

## 關鍵觀點
Pencil.dev 的核心價值在於打破設計與開發的鴻溝——設計師在畫布上調整元件，AI 代理即時理解設計意圖並生成符合專案標準的前端程式碼。搭配 Gemini CLI 的免費額度，整個工作流可以完全零成本。目前免費但預計未來會轉向付費模式。
