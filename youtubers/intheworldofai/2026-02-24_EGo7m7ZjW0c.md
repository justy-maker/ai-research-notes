## 📌 核心觀點

Google AI Studio 正進行最大規模的基礎設施升級，改由 **Antigravity**（Google 的 Agentic IDE 框架）驅動，將 Studio 從簡單的模型試玩場轉變為真正的動態編碼環境。

## 📝 重點功能

### Antigravity 整合
- AI Studio 新增「Build Mode / Asia Mode」，可直接在平台內建構生產級應用
- 支援多人協作（Multiplayer Collaboration）
- 自動引入專業工具如 Lucid React、Frame Motion，應用從第一天就接近部署就緒

### 框架支援
- 目前支援：React、Next.js、Angular
- 測試中：XR Blocks 等其他框架

### 新功能
- **發布功能**：可直接發布到各平台
- **第三方整合**：Slack、Twilio 等服務
- **版本控制**：可回溯到之前的 checkpoint
- **內建身份驗證**：預期將與 Firebase 整合，提供資料庫即時同步和完整認證系統
- **Secrets 管理**：安全儲存 API 金鑰等機密資訊

### 免費 Gemini API
- 可免費使用 Gemini 全套工具：影片生成模型、Google Maps 資料、圖片生成模型等
- 搭配 Gemini 3.1 Pro 可建構各類應用（有人用它做了雷射槍遊戲）
- 所有程式碼可下載或直接推送到 GitHub
