## 📝 核心摘要

使用 **Nano Banana**（圖像生成模型）結合 **Gemini 3** 實現專業級 UI 設計工作流。從生成 mockup 到提取素材，再到代碼實現的完整流程。

---

## 🎯 核心工作流

### 1. 生成高分辨率素材
```
上傳模型 → 提示提取 3D 物體 → 改為 4K 分辨率 → 獲得高質量素材
```

### 2. 持續調整而非重新生成
> 如果生成結果接近目標，**不要刪除重新生成**，而是持續給提示調整

例如：「移除 UI 元素部分，我們會用代碼實現」

### 3. 進階：視差滾動 3D 素材
使用 **Replicate** 將圖片轉換為帶視差效果的 3D 素材：
- 上傳圖片
- 生成浮動 3D 素材
- 嵌入網站做滾動動畫

---

## 🛠️ Super Design 工作流

### 計劃模式功能
- 貼圖 UI 模型 + 提示詞
- 添加產品風格、設計系統上下文
- 上傳 Logo 和素材
- 開啟**視覺模式**讓 AI 看到圖片

### 實際操作步驟

1. **開啟計劃模式** → 發送設計需求
2. **AI 提問澄清** → 確定設計風格（如「跳皮、大膽」）
3. **生成圖像 mockup** → 多版本選擇
4. **修正問題**：
   - Logo 不對？→ 複製 Logo 重新提示
   - 提示：「請把 logo 換成我們的，並保持 UI 比例不變」

---

## 💡 代碼實現技巧

### 簡單 UI
直接進入**構建模式** → 複製 UI

### 複雜 UI
1. 先做計劃：「分析難點，制定計劃解決每個難點」
2. 要求：添加難度、重要節點、替代方案
3. 切換到構建模式實現

### 提取 3D 圖像資源
```
選中圖像 → 提示提取 3D 圖像資源 → 用於網站實現
```

### 去背景工具
使用 Replicate 的去背景工具：
- 上傳圖像 → 點擊運行
- 成本：~1 美元處理 2000 張圖片

---

## 🔄 AI 審查與改進

使用 Nano Banana **審查和評價設計**：
1. 選擇設計
2. 點擊「建議改進」
3. Gemini 3 輸出潛在改進建議
4. 將建議發回編碼代理繼續改進
