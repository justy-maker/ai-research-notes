## 摘要

本集涵蓋多個 AI 領域重要更新：

### Step 3.5 Flash 模型發布
- 專為高速低成本推理打造的開放模型
- 採用稀疏混合專家（MoE）架構：每個 Token 僅激活 110 億參數，總參數量 1960 億
- 支持本地硬件：Apple M4 Max、AMD、NVIDIA 設備
- 推理速度：在 NVIDIA Hopper GPU 上最高約 350 Token/秒
- 提供 INT-4 GGUF 量化格式，上下文窗口最高 256K Token

### MISPU 強化學習框架
- 專為提升長序列訓練穩定性設計
- 解決訓練與推理不匹配、策略漂移問題
- 採用階段感知價值自舉和路由制性度監控技術

### OpenSeal（原 ModelBot/CloudBot）事件
- 由歐洲開發者 Pete Steinberger 開發
- 功能：可接入郵箱、通訊應用、日曆、瀏覽器和本地文件
- 能整理收件箱、發送簡報、辦理航班、通過 WhatsApp/iMessage/Discord 發消息
- GitHub 開源，但配置需要技術門檻
- **安全警告**：本地運行擁有 Shell 權限的 AI 代理有巨大風險

### Modelbook 數據調查
- WIZ 調查發現：約 150 萬個註冊智能體僅對應約 1.7 萬名人類所有者（比例 88:1）
- 許多 AI 發帖實際由人類協助（角色扮演或手動操作）
- 數據泄漏事件：約 150 萬個 API 令牌、約 3.5 萬個郵箱地址被意外暴露
- WIZ 披露後，Modelbook 團隊數小時內封堵漏洞

### 核心問題
- 這些實驗性智能體生態發展太快，安全措施落後
- 我們是在打造終極生產力工具，還是在構建難以掌控的系統？
