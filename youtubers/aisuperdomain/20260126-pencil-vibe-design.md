## 核心論點

Pencil 是 AI 驅動的 UI 設計畫布工具，自然語言描述即可生成專業級 UI 界面，原生支持 Claude Code 自動配置 MCP，程序員終於不用求設計師了。

## 重點整理

### 核心功能
- 自然語言描述生成專業級 UI
- AI 同時生成多個設計方案，風格高度統一
- 原生支持 Claude Code，自動配置 MCP
- 支持 shadcn UI 等多種組件庫

### 實戰演示
1. **Apple Music 風格播放器**：從零創建 9 個完整頁面
2. **與 Google Stitch 對比**：效果優劣分析
3. **Sticky Note 微調**：局部精細調整技巧

### 進階功能
- Claude Code 一鍵生成 React + Tailwind 代碼
- Figma 設計稿複製粘貼導入
- JSON 格式保存與 Git 版本管理

### 使用場景
- 快速原型設計
- 前端開發設計稿
- 設計到代碼一體化流程
