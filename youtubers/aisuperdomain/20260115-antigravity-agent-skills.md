## 核心論點

Google Antigravity IDE 正式支持 Agent Skills，這是繼 OpenAI Codex 之後又一 AI 巨頭加入 Skills 陣營。Prompt 是臨時指令，Skills 才是長期資產。

## 重點整理

### Agent Skills 概念
- **Prompt**：臨時指令，用完即棄
- **Agent Skills**：長期資產，可複用的業務手冊
- 本質是 AI 專用業務手冊

### 安裝配置
- 支持全局路徑或項目路徑
- 可從 Anthropic 官方 Skills 倉庫安裝
- 手動創建 Code Review Skill

### Antigravity 獨家功能
- 自動調用 Nano Banana 模型生成配圖
- 這是 Codex 和 Claude Code 都不具備的能力

### 實戰演示
1. **前端設計 Skill**：創建咖啡店落地頁 + 自動配圖
2. **UI UX Pro Max**：最強 UI 設計 Skills
   - 支持 React/Next.js/Swift 等多技術棧
   - 擬物化風格創建精美 To-Do List
