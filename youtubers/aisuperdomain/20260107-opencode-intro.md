## 核心論點

OpenCode 是可完美替代 Claude Code 的開源 AI 編程工具，支持免費使用多款大模型，完整 LSP 支持解決 Claude Code 閃爍問題，Token 消耗和文件變更一目了然。

## 重點整理

### 免費模型支持
- GLM-4.7
- MiniMax M2.1
- Grok Code
- 支持 Claude 訂閱賬號登入

### 核心優勢
- **開源 MIT 許可**
- **多模型切換**
- **插件系統**
- **完整 LSP 支持**：解決閃爍問題
- **精緻 TUI 界面**：Token 消耗一目了然

### 快速上手
1. Homebrew 安裝
2. `/init` 命令生成 AGENTS.md
3. opencode.json 配置 LSP

### 實戰演示
- 為 iOS 背單詞 App 添加「判斷題」和「多選題」功能
- 結合 OpenSpec 規格驅動開發
- 全程零 Bug 通過測試
