# 垂直AI代理可能比SaaS大10倍（中文翻譯）

## 影片資訊
- 原標題：Vertical AI Agents Could Be 10X Bigger Than SaaS
- 連結：https://www.youtube.com/watch?v=ASABxNenD_U
- 頻道：Y Combinator

## 核心觀點
1. **垂直AI代理的巨大潛力：** 影片認為垂直AI代理將會是一個巨大的市場，甚至可能比SaaS更大，有望產生市值超過3000億美元的公司。
2. **SaaS的成功經驗：** 影片透過回顧SaaS的發展歷程，指出SaaS的成功是因為解決了特定行業或職能的痛點，並提供了比通用軟體更好的使用者體驗。
3. **LLM的類比：** 影片將大型語言模型（LLM）的出現類比為SaaS發展初期的XML HTTP request技術，認為LLM將會帶來新的運算範式，並創造新的機會。
4. **創業策略：** 影片建議創業者避免在通用AI應用領域與大型公司競爭，而是專注於開發垂直AI代理，解決特定行業或職能的具體問題。
5. **團隊建設的轉變：** 影片指出，隨著AI技術的發展，新創公司在達到產品市場契合後，可能需要更多擅長LLM的軟體工程師，而不是傳統的銷售或客服人員，以實現自動化並降低成本。

## 詳細內容摘要
影片主要探討了垂直AI代理的巨大潛力，並將其與SaaS的發展歷程進行類比，以說明其發展前景。影片首先指出，SaaS在過去20年中一直是矽谷投資的重點，產生了超過300家獨角獸公司。SaaS的成功是因為它解決了特定行業或職能的痛點，並提供了比通用軟體更好的使用者體驗。

影片接著回顧了SaaS發展的歷史，指出XML HTTP request技術的出現是SaaS發展的催化劑，它使得在瀏覽器中構建豐富的網路應用程式成為可能。影片將大型語言模型（LLM）的出現類比為SaaS發展初期的XML HTTP request技術，認為LLM將會帶來新的運算範式，並創造新的機會。

影片進一步分析了SaaS公司成功的模式，指出創業者可以選擇三條路徑：開發大眾消費產品、開發非預期的消費產品，以及開發B2B SaaS產品。影片認為，在通用AI應用領域，大型公司更有可能勝出，因此創業者應該專注於開發垂直AI代理，解決特定行業或職能的具體問題。

影片還討論了團隊建設的轉變，指出隨著AI技術的發展，新創公司在達到產品市場契合後，可能需要更多擅長LLM的軟體工程師，而不是傳統的銷售或客服人員，以實現自動化並降低成本。影片認為，未來可能會出現只有10名員工的獨角獸公司，他們主要負責編寫提示詞和評估AI模型的輸出。

影片最後強調，現在是垂直AI代理發展的早期階段，創業者應該抓住機會，開發能夠解決特定問題的AI代理，並利用AI技術來降低成本和提高效率。
