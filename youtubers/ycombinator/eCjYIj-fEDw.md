# What Boris Cherny Learned From Building Claude Code
- 頻道：Y Combinator
- 連結：https://www.youtube.com/watch?v=eCjYIj-fEDw
- 日期：2026-02-21

## 重點摘要
- Boris Cherny 創造 Claude Code 時連續三個月沒休假，包括週末和每晚都在工作
- 一開始還不確定是否有用，因為當時模型還不能真正寫程式
- 終端（Terminal）本來只是起點，沒想到成為了最終形態
- **給 LLM 創業者的建議**：不要為今天的模型構建產品，要為 6 個月後的模型構建
- 關注模型目前「還不太擅長」的前沿領域，因為模型很快就會變好
