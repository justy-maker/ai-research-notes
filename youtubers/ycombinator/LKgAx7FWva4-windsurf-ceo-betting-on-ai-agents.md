# Windsurf CEO：押注 AI Agents（中文翻譯）

## 影片資訊
- 原標題：Windsurf CEO: Betting On AI Agents
- 連結：https://www.youtube.com/watch?v=LKgAx7FWva4

## 核心觀點（3-5 個重點）

1.  **持續創新是新創公司的生存之道：** 新創公司必須不斷證明自己，每個洞見都會隨著時間貶值。如果沒有持續的創新和執行，公司就會慢慢走向衰亡。
2.  **擁抱快速轉型：** 當市場環境發生變化，原有的假設被證明是錯誤的，公司必須迅速轉型。即使已經取得一定的成功，如果看不到擴展的潛力，也應該果斷改變方向。
3.  **團隊熱情至關重要：** 在轉型過程中，選擇一個能激發團隊熱情的方向至關重要。如果團隊對新的方向沒有熱情，即使這個方向看起來很有價值，也很難成功。
4.  **結合不合理的樂觀主義和毫不妥協的現實主義：** 新創公司需要同時具備這兩種看似矛盾的特質。不合理的樂觀主義能驅動行動，而毫不妥協的現實主義則能確保在事實發生變化時及時調整方向。
5.  **AI Agents 的未來：** 影片暗示了開發者角色將會擴展到「構建者」，軟體開發將會變得非常普及。Windsurf 正在押注 AI Agents 的未來，並致力於讓開發者能夠更輕鬆地構建和部署 AI 應用。

## 詳細內容摘要（500-800 字）

這段影片採訪了 Windsurf 的共同創辦人兼 CEO Verun，討論了 Windsurf 的創業歷程，以及他們如何從 GPU 虛擬化公司轉型為 AI 程式碼輔助工具公司。

Windsurf 最初是一家名為 Exofunction 的公司，專注於 GPU 虛擬化，旨在簡化深度學習工作負載的運行。他們看到了深度學習在各個行業的潛力，並為自動駕駛和 AR/VR 等領域的公司提供 GPU 資源。然而，在 2022 年中期，隨著 Transformer 模型的興起，他們意識到原有的商業模式可能會被顛覆。他們預見到，如果大家都使用相同的 Transformer 模型，GPU 基礎設施提供商將會被商品化。

因此，Verun 和他的共同創辦人決定進行一次大膽的轉型。他們在一個週末的時間裡，經過深入的討論，決定將公司轉向 AI 程式碼輔助工具領域。他們是 GitHub Copilot 的早期使用者，並認為這個產品只是冰山一角，AI 在程式碼輔助方面還有更大的潛力。

這個轉型對他們來說是一個巨大的賭注，因為他們當時已經籌集了 2800 萬美元的資金，並且已經實現了數百萬美元的收入。然而，他們認為，如果他們看不到擴展的潛力，即使現在做得還不錯，也應該果斷改變方向。

為了確保轉型成功，他們選擇了一個能激發團隊熱情的方向。他們認為，如果團隊對新的方向沒有熱情，即使這個方向看起來很有價值，也很難成功。

在轉型後，Windsurf 迅速開發了一個名為 Kodium 的 VS Code 擴充功能，並在兩個月內發布。最初的版本不如 GitHub Copilot，但它是免費的。他們很快改進了他們的模型訓練基礎設施，並開始訓練自己的模型。在幾個月內，他們的模型在某些方面甚至超越了 GitHub Copilot，例如能夠填補程式碼中間的空白。

Windsurf 的快速轉型和創新吸引了許多大型企業客戶，例如 Dell 和 JP Morgan Chase。這些公司希望在安全環境中運行程式碼輔助工具，並將其個人化到自己的私有資料。

Verun 強調，新創公司需要不斷證明自己，每個洞見都會隨著時間貶值。如果沒有持續的創新和執行，公司就會慢慢走向衰亡。他還提到，新創公司需要同時具備不合理的樂觀主義和毫不妥協的現實主義。不合理的樂觀主義能驅動行動，而毫不妥協的現實主義則能確保在事實發生變化時及時調整方向。

影片最後暗示了開發者角色將會擴展到「構建者」，軟體開發將會變得非常普及。Windsurf 正在押注 AI Agents 的未來，並致力於讓開發者能夠更輕鬆地構建和部署 AI 應用。

## 關鍵語錄（2-3 句原文+翻譯）

*   "Every single insight that we have is a depreciating insight. If we don't continually have insights that we are executing on, uh we are just slowly dying."
    *   「我們擁有的每一個洞見都會貶值。如果我們沒有持續的洞見並加以執行，我們就會慢慢走向死亡。」
*   "You need this irrational optimism because if you don't have the optimism, you just won't do anything... and you need uncompromising realism, which is that when the when the facts change, you actually change your mind."
    *   「你需要這種不合理的樂觀主義，因為如果你沒有樂觀主義，你什麼都做不了……而且你需要毫不妥協的現實主義，也就是說，當事實發生變化時，你實際上會改變你的想法。」

## 我的評論（對創業者/開發者的啟發）

Verun 的故事對創業者和開發者來說是一個強大的啟示。它展示了快速轉型、擁抱變化和持續創新的重要性。即使已經取得一定的成功，也不應該安於現狀，而應該不斷尋找新的機會和挑戰。

對於開發者來說，這個故事也強調了學習新技術和適應市場變化的重要性。AI 正在迅速改變軟體開發的格局，開發者需要不斷學習新的技能，才能在這個快速變化的世界中保持競爭力。

此外，Verun 的故事也提醒我們，團隊的熱情和動力是成功的關鍵。選擇一個能激發團隊熱情的方向，並確保團隊成員都對這個方向充滿信心，是至關重要的。

最後，這個故事也強調了不合理的樂觀主義和毫不妥協的現實主義的重要性。創業者需要同時具備這兩種看似矛盾的特質，才能在充滿挑戰的創業道路上取得成功。
