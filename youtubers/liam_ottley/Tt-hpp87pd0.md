# Did Claude Just Replace Video Editors? (Remotion Breakdown)

## 一句話總結

影片探討了 Remotion 與 Claude Code 的結合如何透過自然語言提示，以 AI 驅動的方式簡化影片編輯流程，使開發者能更輕鬆地創建影片。

## 重點

- **Remotion 整合 Claude Code**：讓使用者可以使用自然語言生成和修改影片
- **簡單英文描述**：用簡單的英文描述想要的影片內容，由 Claude Code 解釋指令並生成 React/TypeScript 程式碼
- **宣告式模型**：將手動影片編輯轉變為宣告式模型，由 Claude 作為「影片邏輯引擎」
- **多種應用場景**：可用於生成大量個人化影片、數據視覺化、自動化社群媒體內容和動態產品展示
- **製作流程**：自然語言提示 → Claude 生成 Remotion 程式碼 → 預覽 → 渲染 MP4
