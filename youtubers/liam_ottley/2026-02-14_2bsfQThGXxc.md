### 核心概念

- **工作區（Workspace）**：包含執行特定任務所需的檔案、資料夾、指令、腳本和情境脈絡的資料夾。將任務按功能劃分至不同工作區，有助於保持組織性。

- **指令（Commands）**：可重複使用的提示或指示，用於自動化工作流程。通常以自然語言編寫，儲存在 Markdown 檔案中。

- **技能（Skills）**：類似外掛程式，可擴展工作區功能，提供額外工具。可從技能市集取得。

- **腳本（Scripts）**：由 Claude Code 建立的程式碼片段，用於執行特定動作（如從網路提取資料）。使用者無需自行編寫程式碼。

- **情境脈絡（Context）**：關於使用者、工作和目標的相關資訊。提供充分的情境脈絡可讓 Claude Code 更準確地理解意圖。

- **/prime 指令**：在 Claude Code 啟動時載入所有相關情境脈絡，確保充分理解工作區內容。

- **情境堆疊（Context Stacking）**：將不同層級的資訊堆疊在一起（目標、業務、策略、數據等）。

### 實用技巧與建議

1. **建立明確的工作區**：根據不同功能或領域建立獨立工作區
2. **將工作流程分解為步驟**：將複雜工作流程分解為明確步驟，編寫成指令
3. **撰寫可重複使用的指令**：節省時間並確保任務執行一致性
4. **定期更新情境脈絡**：隨工作變化定期更新
5. **利用腳本自動化任務**：自動化資料提取、處理等任務
6. **使用 /create plan 和 /implement 指令**：讓 Claude Code 協助規劃和實施新工作流程
7. **不要把 Claude Code 當作聊天機器人**：創建可重複使用的命令
8. **總是給予情境**：每個新環節使用 `/prime` 命令提供背景資訊

### 總結與關鍵收穫

- **Claude Code 是強大的自動化工具**：可將重複性任務自動化，節省時間精力
- **情境脈絡至關重要**：充分的情境讓 Claude Code 更準確理解意圖
- **工作流程建立是關鍵**：分解步驟、編寫指令讓 Claude Code 更好執行任務
- **持續學習和探索**：隨 Claude Code 發展，持續探索新功能和技巧

透過情境堆疊和工作流程，Claude Code 可以成為個人數位助理，協助完成各種任務。這是一份實用且易於理解的入門指南，無論初學者還是有經驗的使用者都可從中獲益。
