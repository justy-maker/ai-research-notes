## 📌 核心觀點

OpenClaw/ClawHub 的出現讓 SaaS 創業門檻崩塌。過去需要 5-20 萬美元和 6-12 個月才能建構的 SaaS，現在一個人一個週末就能做出來。但 Liam 認為對大多數人來說，這是「在沙子上蓋房子」，AI 代理/諮詢的商業模式更穩固。

## 📝 五類 Skill-based 商業模式

### 1. 純 Prompt 技能（專業知識文件化）
- 就是一個文字檔，教 AI 怎麼做某件事
- 例：合約審查、寫作框架、研究流程
- 護城河幾乎為零，賣 $10-50
- 適合當引流或免費素材

### 2. 工具類技能（Utility Skills）
- 包裝一段實際執行任務的腳本
- 例：高品質 YouTube 字幕提取器、網頁爬蟲、PDF 解析器
- 賣點是「它能用，壞了有人修」→ $5-15/月

### 3. API 整合技能（橋接服務）
- 教 AI 如何正確使用現有工具（如 HubSpot CRM）
- 價值在整合邏輯，不只是呼叫 API
- $20-100 一次性或訂閱

### 4. 後端服務技能（Skills as a SaaS）⭐
- 你在伺服器上跑服務，Skill 教 AI 如何連接
- 例：公司情報服務（自動爬取 LinkedIn、Crunchbase、新聞，合成報告）
- 技術門檻：100-200 行程式 + $10-20/月主機 + 一頁式 Landing Page + Stripe
- Skill 指令公開但沒你的伺服器就沒用 → 護城河強
- **$9-50/月，100 用戶 × $19 = $1,900/月**

### 5. 專有資料技能（護城河最強）
- 以獨有資料建構 Vector Database，AI 查詢後回傳答案
- 用戶得到洞察但看不到原始資料
- 例：軟體市場定價情報
- $19-200/月

## ⚠️ Liam 的誠實忠告
- OpenClaw 是第一版，OpenAI/Google/Anthropic 一定會推競品
- 現在贏的人是長年深耕 AI 的頂級工程師
- 企業暫時無法使用（安全合規不到位）
- **對多數人：AI 代理/諮詢（在岩石上蓋房）比追新平台（在沙上蓋房）更穩**
- 學會 AI 審計、無程式碼自動化、培訓導入 → 未來十年都有價值
