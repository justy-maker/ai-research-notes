# Claude Code - Full Tutorial for Beginners
- **頻道**: Tech With Tim
- **日期**: 2026-02-28
- **影片連結**: https://www.youtube.com/watch?v=ntDIxaeo3Wg

## 重點摘要
- **AI 驅動的開發助理**：Claude Code 是一款 AI 編碼助理工具，開發者能透過終端機（Terminal）下達指令，讓 AI 協助完成軟體設定、程式碼撰寫與專案建構等任務。
- **完整的安裝與設定教學**：影片從零開始，詳細介紹了安裝 Claude Code 的前置需求（如 Git），並引導使用者完成安裝過程，以及將工具與個人 GitHub 帳號連結的步驟。
- **互動式的開發流程**：此工具的核心模式是「指令-計畫-批准」。使用者下達開發需求後，AI 會提出執行計畫與程式碼變更，並等待使用者手動批准後才會實際執行，確保開發者保有最終控制權。
- **實戰範例演練**：影片透過建立一個井字遊戲（Tic-Tac-Toe）的 React 網站作為範例，實際展示如何使用 Claude Code 來初始化專案、建立與更新檔案，並執行建構指令。

## 關鍵觀點
- **人機協作新模式**：Claude Code 將 AI 轉化為一個整合在本地開發環境中的代理（Agent），它不僅僅是產生程式碼片段，更能理解專案結構並執行檔案操作、運行指令，實現更深度的開發自動化。
- **賦予開發者監督權**：其「手動批准」的機制是關鍵，它在享受 AI 帶來的高效率同時，也避免了 AI 失控或產生非預期變更的風險，讓開發者能安心地將其導入工作流程。

## 一句話總結
這是一份專為初學者設計的完整教學，說明如何安裝、設定並使用 Claude Code 這款 AI 編碼工具，透過與 AI 協作來自動化完成軟體專案開發。
