## 重點整理

### 什麼是 ML 工程師？
- 建立和部署機器學習系統
- 結合軟體工程和資料科學
- 高需求、高薪資的職位

### ML 工程師市場數據
- 就業需求持續成長
- 平均薪資範圍
- 產業分布

### 學習路線圖

**#1 Python 基礎**
- 語法和資料結構
- 物件導向程式設計
- 常用函式庫

**#2 電腦科學基礎**
- 演算法和資料結構
- 時間複雜度分析
- 基本數學概念

**#3 工具使用**
- Git 版本控制
- 命令列操作
- 開發環境設定

**#4 資料處理**
- NumPy 和 Pandas
- 資料清理和轉換
- 資料視覺化

**#5 互動學習**
- Jupyter Notebooks
- 實驗和探索
- 快速迭代

**#6 核心 ML 演算法**
- 監督式學習
- 非監督式學習
- 模型評估

**#7 深度學習**
- 神經網路基礎
- TensorFlow / PyTorch
- CNN、RNN 等架構

**#8 強化學習**
- 基本概念
- 應用場景
- 實作練習

**#9 LLMs（大型語言模型）**
- Transformer 架構
- Fine-tuning 技術
- 實際應用

**#10 MLOps**
- 模型部署
- 監控和維護
- CI/CD for ML
