## 重點整理

### 什麼是 Matplotlib？
- Python 最流行的資料視覺化函式庫
- 資料科學和機器學習的必備工具
- 可生成出版等級的圖表

### 安裝與設定
- 使用 pip 安裝：`pip install matplotlib`
- 通常搭配 NumPy 和 Pandas 使用

### 基本匯入
```python
import matplotlib.pyplot as plt
import numpy as np
```

### Matplotlib 架構
- Figure：整個圖形視窗
- Axes：實際的繪圖區域
- 理解架構有助於進階客製化

### 基本折線圖
- `plt.plot()` 繪製基本線條
- 設定 x 軸和 y 軸數據

### 樣式客製化
- 線條顏色、粗細、樣式
- 標籤、標題、圖例
- 網格線設定

### 散點圖 (Scatter Plot)
- 展示數據點分布
- 適合相關性分析

### 長條圖 (Bar Chart)
- 比較不同類別的數值
- 水平和垂直版本

### 直方圖 (Histogram)
- 展示數據分布
- bins 參數控制分組數量

### 子圖 (Subplots)
- 在一個 Figure 中放置多個圖表
- 使用 `plt.subplots()` 建立

### Pandas 整合
- 直接從 DataFrame 繪圖
- 自動處理索引和標籤

### 進階客製化
- 調整字體、顏色主題
- 添加註釋和箭頭

### 圖表樣式
- 內建多種樣式主題
- 使用 `plt.style.use()` 切換

### 圓餅圖 (Pie Chart)
- 展示比例分布
- 可加入百分比標籤

### 儲存圖表
- `plt.savefig()` 儲存為圖片
- 支援 PNG、PDF、SVG 等格式
