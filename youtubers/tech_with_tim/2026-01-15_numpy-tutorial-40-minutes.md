## 重點整理

### 什麼是 NumPy？
- Python 中用於數值計算的核心函式庫
- 提供高效的多維陣列運算
- 是 Pandas、TensorFlow、PyTorch 等的基礎

### 為什麼使用 NumPy？
- 比原生 Python 列表快數十倍
- 記憶體使用更有效率
- 提供大量數學函數

### 安裝與設定
- 使用 pip：`pip install numpy`
- 標準匯入：`import numpy as np`

### 建立 NumPy 陣列
- `np.array()` 從列表建立
- `np.zeros()`, `np.ones()` 建立特定值陣列
- `np.arange()`, `np.linspace()` 建立數列

### NumPy 資料型別
- int32, int64, float32, float64 等
- 可指定 dtype 參數
- 記憶體效率考量

### 多維陣列
- 建立 2D、3D 陣列
- 理解 shape 屬性
- reshape 改變形狀

### 陣列屬性
- shape：陣列維度
- dtype：資料類型
- size：元素總數
- ndim：維度數量

### 索引與切片
- 基本索引：`arr[0]`, `arr[1, 2]`
- 切片：`arr[0:5]`, `arr[:, 1]`
- 布林索引：`arr[arr > 5]`

### 陣列運算
- 元素級運算：+, -, *, /
- 廣播機制（Broadcasting）
- 通用函數：np.sqrt, np.exp 等

### 陣列操作
- 合併：concatenate, vstack, hstack
- 分割：split, vsplit, hsplit
- 轉置：transpose, T

### 維度與軸
- 理解 axis 參數
- 沿特定軸運算

### 統計運算
- mean, std, var
- min, max, sum
- argmin, argmax

### 線性代數
- 矩陣乘法：np.dot, @
- 行列式、特徵值等

### 實用方法
- where, unique, sort
- 隨機數生成：np.random
