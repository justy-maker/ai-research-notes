## 核心論點
CANN（Compute Architecture for Neural Networks）是華為推出的 AI 異構計算架構，作為 AI 框架與硬件處理器之間的橋樑，通過深度開放、創新編程語言和運行時開放，推動中國 AI 產業創新。

## 重點整理

### CANN 定位
- 全名：Compute Architecture for Neural Networks
- AI 芯片界的「超級翻譯官」
- 連接上層 AI 框架（PyTorch、TensorFlow）與底層昇騰 AI 處理器

### 三大核心特性

#### 1. 深度開放
- 基礎算子、融合算子全部開源
- 提供豐富的算子模板庫
- 降低創新門檻，不再是大廠專利

#### 2. OSAND C2.0 編程語言
- 原生支援 C 和 C++
- 支援捲積運算和通算融合
- 大模型訓練時通信和計算可同時進行
- 140+ 基礎 API + 80+ 高級 API
- 簡化高性能算子開發難度

#### 3. 運行時深度開放
- 全套資源管理、模型推理、集合通信功能
- 支援自定義底層接口
- 輕量化圖能力
- 支援單機多卡、多機多卡超大規模訓練推理

### 生態建設
- 開發者從 1000 人增長到 6000 人
- 清華、哈工大等頂級學府合作
- 昇騰 API 算子挑戰賽
- 覆蓋互聯網、金融、運營商、基礎模型等行業

### 三個極致
1. **極致性能**：硬件親合的深度優化
2. **極簡開發**：統一編程接口降低門檻
3. **開放生態**：無縫對接各種 AI 框架
