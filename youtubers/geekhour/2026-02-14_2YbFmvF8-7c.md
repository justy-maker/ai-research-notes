## 影片摘要

本影片介紹 **Index TTS 2**，一款由 B站推出的開源免費語音生成模型，只需要一句話就可以複製你的聲音。

### 核心功能

1. **聲音克隆**：只需一小段語音樣本，就能生成與原聲音色相似的語音
2. **情感模擬**：能模擬出不同的情感特點（開心、生氣、悲傷、驚喜等）
3. **多語言支援**：支援包括英文、日文在內的多種語言，甚至國內方言也能準確合成

### 硬體要求

- 顯存：約 8G 左右即可運行
- 一般的 Mini 級顯卡都可以安裝使用

### 應用場景

- 影視配音
- 有聲書製作
- 語音助手
- 播客等

### 使用方法

1. **Web 介面操作**：通過瀏覽器進行操作
2. **上傳語音樣本**：準備一小段語音樣本
3. **輸入文字內容**：在介面輸入想要生成的文字
4. **生成語音**：點擊生成按鈕即可

### 進階功能

- **音色 + 情感分離**：可分別使用音色參考音頻和情感參考音頻進行語音合成
- **情感權重調節**：控制情感參考對最終語音的影響程度（0-1.0）
- **情感參數調節**：可分別調節開心、生氣、悲傷等多種情感參數

### 安裝步驟

1. 安裝 Git 和 Git LFS
2. 克隆 Index TTS 代碼倉庫到本地
3. 安裝 Python 環境（建議 3.10 以上版本）
4. 安裝 pip 和 UV（必須使用 UV 來安裝和管理依賴）
5. 使用命令安裝所需依賴（國內建議使用阿里雲或清華鏡像源）
6. 下載模型文件（使用魔搭 Modelscope）
7. 運行 `uv run webui.py` 啟動 Web 介面
8. 瀏覽器訪問 `localhost:7860`

### 注意事項

- **必須使用 UV** 來安裝和管理依賴，使用 Conda 或 pip 可能出現各種問題
- Windows 環境下安裝 DeepSpeed 可能會出現問題，需多嘗試幾次
- 影片提供整合包，可直接下載解壓使用

### 總結

Index TTS 2 在聲音克隆和情感表達方面表現優秀，是影視配音、有聲書製作等領域的實用工具。開源免費、硬體要求低，非常適合個人用戶嘗試使用。
