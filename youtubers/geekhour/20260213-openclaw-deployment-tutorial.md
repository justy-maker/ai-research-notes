## 📌 重點摘要

- **OpenClaw 爆紅**：一週暴漲 100 萬+ Star，已超越 DeFi，正在追趕 React 和 Linux 等頂級開源專案
- **核心功能**：部署在自己設備上的 AI 助手，可透過釘釘、QQ 等聊天工具 7x24 小時互動
- **國內部署方案**：推薦使用「無影雲電腦」一鍵部署，每月 49 元，避免環境配置和網路問題
- **Skills 整合**：預裝代碼生成、圖片生成、文件管理、網頁瀏覽等常用功能
- **實測展示**：成功用 OpenClaw 從零搭建一個科技博客網站（用戶註冊、登入、文章發布、評論等功能）

## 💡 關鍵洞見

OpenClaw 不只是簡單的 AI 助手，更像是一個可以不斷擴展和定制的平台。它的工作原理是：監聽聊天工具的消息 → 轉發到設備上的 OpenClaw 程式 → 調用底層工具和 Skills 執行任務 → 返回結果。功能上限很高，可玩性非常強。

## 🔧 部署步驟（無影雲電腦方案）

1. **購買雲電腦**
   - 登入無影雲電腦網頁版
   - 選擇含黃金月卡和 2000 核時包的套餐（活動價 49 元/月）

2. **下載客戶端**
   - 支援 Mac、Windows、Android、iOS
   - 記得關閉「斷連定時關機」功能，保持 24 小時在線

3. **配置百鏈大模型 API**
   - 註冊百鏈帳號
   - 在「應用開發」→「密鑰管理」創建 API Key
   - 貼到 OpenClaw 配置面板測試連接

4. **配置釘釘機器人**
   - 釘釘開發者平台創建應用
   - 添加機器人能力，選擇 Stream 模式
   - 複製 ClientID 和 ClientSecret 到配置面板
   - 將機器人添加到群聊

5. **開始使用**
   - 在群內 @ 機器人下達指令
   - 例如：「幫我安裝 Java」、「用現代技術棧搭建一個科技博客網站」
