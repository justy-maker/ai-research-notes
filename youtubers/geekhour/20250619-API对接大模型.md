## 重點整理

### API 對接的優勢
- 直接使用雲服商的大模型能力
- 無需購買昂貴硬件設備
- 不需要關心底層實現細節
- 可選擇多種大模型

### 主要大模型 API 服務商
- **官方廠商**：OpenAI、DeepSeek、通義千問
- **第三方平台**：矽流動、端腦雲

### 端腦雲特點
- 上線滿血版 DeepSeek R1 模型
- 對接方式與官方完全一致
- 價格是主流平台的八折不到
- 適合個人開發者和小團隊

### Cherry Studio 使用教程

#### 配置步驟
1. 下載安裝 Cherry Studio
2. 添加供應商（端腦雲/Siliconflow）
3. 配置 API 密鑰
4. 添加模型（DeepSeek R1）
5. 配置默認助手

#### 注意事項
- API 密鑰要妥善保管，不要洩露
- 模型 ID 必須完全正確
