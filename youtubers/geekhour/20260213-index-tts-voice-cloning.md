## 重點摘要

### 什麼是 Index TTS

Index TTS 是由 B 站推出的一款開源免費語音生成模型，只需要一小段語音樣本（一句話），就能克隆出該聲音並生成任意文字內容的語音。

**特色功能：**
- 聲音克隆：準確還原不同人物的音色
- 情感模擬：能模擬憤怒、驚喜、悲傷等情感特點
- 多語言支援：包括中文、英文、日文，甚至部分方言
- 硬體需求低：8G 顯存即可運行，一般民用級顯卡都可使用

### 三種使用模式

1. **基本聲音克隆**
   - 上傳一段語音樣本
   - 輸入想要生成的文字
   - 系統生成帶有該聲音特點的語音

2. **音色 + 情感分離控制**
   - 可分別上傳音色參考音頻和情感參考音頻
   - 生成的語音會保留音色特點，但帶有另一段音頻的情感
   - 可調整「情感權重」參數控制影響程度（0-1.0）

3. **情感參數調節**
   - 可手動調節開心、生氣、悲傷、驚喜等多種情感參數
   - 精確控制生成語音的情感強度

### 安裝步驟

1. **安裝 Git 和 Git LFS**
   - Git LFS 用於管理大型檔案
   - 安裝後執行 `git lfs install`

2. **克隆專案代碼**
   ```bash
   git clone [專案地址]
   ```

3. **安裝 Python 環境**
   - 建議 3.10 以上版本

4. **使用 UV 安裝依賴**（重要！）
   - 必須使用 UV 管理依賴，不要用 conda 或其他工具
   - 國內推薦使用阿里雲或清華鏡像源

5. **下載模型文件**
   - 國內推薦使用魔搭（ModelScope）下載

6. **啟動服務**
   ```bash
   uv run webui.py
   ```
   - 開啟瀏覽器訪問 `localhost:7860`

### 應用場景

- 影視配音
- 有聲書製作
- 語音助手
- 播客製作

## 個人筆記

Index TTS 的「情感分離控制」功能很有意思——可以讓一個溫柔女聲帶著憤怒的語氣說話。這對於配音工作來說非常實用，不需要找不同情緒的樣本，只要有基礎音色和情感參考就行。

**注意事項：**
- 一定要用 UV 安裝，用 conda 會出問題
- Windows 安裝 Deep Speed 可能會有些小問題，多試幾次
