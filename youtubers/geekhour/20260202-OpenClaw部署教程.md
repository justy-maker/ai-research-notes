## 分類
AI 工具、開源項目、自動化、雲端部署

## 關鍵詞
OpenClaw、AI 助手、無影雲電腦、釘釘機器人、自動化、7x24 小時、skills

## 核心論點
OpenClaw 是一個可部署在個人設備上的 AI 助手，通過釘釘、QQ 等聊天工具實現 7x24 小時在線，結合 skills 完成各種複雜任務。使用無影雲電腦可以簡化國內用戶的部署難度。

## 重點整理

### OpenClaw 介紹
- GitHub 一週暴漲 100 多萬 Star
- 可部署在個人設備的 AI 助手
- 支援釘釘、QQ、飛書等聊天工具
- 7x24 小時在線服務
- 可配置 skills 完成複雜任務

### 安裝方式
1. 一鍵部署腳本
2. NPM 安裝
3. Mac 應用
4. **推薦**：無影雲電腦一鍵部署

### 無影雲電腦部署優勢
- 不用折騰環境
- 不需購買 Mac mini
- 數據安全
- 活動價：49 元/月
- 預裝 MocBot 環境

### 配置步驟
1. 購買無影雲電腦套餐
2. 下載安裝客戶端
3. 配置百煉大模型 API Key
4. 配置釘釘機器人
   - 創建釘釘應用
   - 添加機器人能力
   - 選擇 Stream 模式
   - 複製 ClientID 和 ClientSecret

### 工作原理
1. 監聽聊天工具消息
2. 轉發到 OpenClaw 程序
3. 調用底層工具和 skills
4. 執行操作並返回結果

### 預裝 Skills
- 代碼生成
- 圖片生成
- 文件管理
- 網頁瀏覽
