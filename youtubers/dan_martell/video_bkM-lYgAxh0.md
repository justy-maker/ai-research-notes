## 🎯 6 大 ChatGPT 技巧

### 技巧 1：升級到 ChatGPT Plus
- 停止省小錢，一個帳號比一頓午餐便宜
- 進階功能是後面技巧的基礎
- 更快、更好整合、更多功能

### 技巧 2：建立角色主提示 (Role Master Prompt)

**為公司每個角色建立完整的「自我介紹」給 AI**

包含：
- 你是誰、公司規模、營收
- 團隊結構、專案、客戶類型
- 產品資訊、一切相關背景

#### 建立步驟：
1. 告訴 GPT：「Interview me to create a master prompt as the CEO of my company」
2. 用語音模式回答所有問題
3. 把結果轉成 Canvas 編輯
4. 存成 PDF 供任何 AI 模型使用

#### 有了 Master Prompt 後 GPT 可以：
- 起草 Playbooks 和 SOPs
- 分析財務資訊
- 壓力測試產品/定價想法
- 建議下一個招聘
- 產生與你目標對齊的策略

> "Sequencing equals success. If the answers aren't aligned with who you are, it can't properly sequence."

### 技巧 3：建立系統提示 (System Prompts)

**系統提示是未來的智慧財產權**

#### 建立步驟：
1. 描述你想要的輸出（加上限制條件）
2. 在 Canvas 中優化輸出
3. 要求 GPT 寫出「能可靠產生這個輸出的詳細系統提示」
4. 在新視窗測試是否一致
5. 存成 PDF 像模板一樣反覆使用

#### 7 個提示關鍵詞：

| 關鍵詞 | 功能 |
|--------|------|
| **Act as [role]** | 設定專業身份或語氣 |
| **Deep research** | 深度搜尋、驗證、交叉引用 |
| **First principles** | 從基本原理分解重建 |
| **Devil's advocate** | 壓力測試、列出風險、反駁 |
| **Constraints first** | 設定時間/預算/工具限制 |
| **Format as** | 指定輸出格式（YAML/JSON/表格/列表） |
| **Verify and cite** | 確認事實並附上引用來源 |

### 技巧 4：用專案資料夾運營公司 (Projects)

**Projects 功能讓 context 永久保存**

- 投資決策、研究、專案各建資料夾
- 上傳所有相關文件：媒體、銷售資訊、報告、email 匯出
- 上傳 Master Prompt 讓回答更精準

#### 核心概念：Compounding Context
- Context 越積累，回答越好
- 同一專案內的每次對話都會利用所有上傳資訊

### 技巧 5：更新自訂指令 (Custom Instructions)

**把 ChatGPT 帳號設定成公司的大腦**

設定 > 個人化 > 自訂指令

設定項目：
- 行為和回應方式
- 風格（簡短、條列式、YAML 格式等）
- 語氣和偏好
- 例：「Always respond in short, concise, bulleted points」

這些設定會永久生效，每次回答都會遵循。

### 技巧 6：建立自訂 GPTs

**不只是提示，是把提示產品化**

把 System Prompt 變成可分享的工具：
- 你自己反覆使用
- 團隊成員可以用
- 不需要給原始提示，給連結就好

#### Dan 的範例：
用 2 小時建立一個系統提示，給它任何人名和主題，它就能產出 2000 字的書籍大綱：
- Sarah Blakeley on parenting
- Gary Vaynerchuk on scaling a media company
- Mr. Beast on building a media department

#### 建立步驟：
1. ChatGPT > GPTs > 右上角 Create
2. 輸入 System Prompt 作為指令
3. 上傳 Master Prompt 到 Knowledge
4. 設定分享權限（任何人/有連結的人）
5. 測試後透過 Slack 推廣到全公司

## 📝 筆記

Dan 這套 ChatGPT 方法論的核心邏輯：

1. **個人化**：Master Prompt 讓 AI 完全了解你的處境
2. **可複製**：System Prompt 讓優秀的輸出可以重複
3. **累積性**：Projects 讓 context 不斷增長
4. **可分享**：Custom GPTs 讓團隊都能使用

最終目標：把 AI 從「工具」變成「團隊成員」，你從「執行者」變成「導演」。

系統提示 = 新時代的智慧財產權。公司估值不只看營收，還要看內部 AI 自動化的程度。
